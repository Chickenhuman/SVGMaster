SVG Master

👋 Welcome to SVG Master!

SVG Master is a 100% free, web-based vector editor.
Draw shapes, paths, and get the SVG code instantly.

  • 🌐 Languages: English, 한글, 日本語 supported.
  • 🚀 No Install: Works directly in your browser.
  • ⌨️ Shortcuts: Press F1 to see useful shortcuts.
  • 📧 Feedback: Found a bug? Check the bottom left sidebar.

* This tool is open source and free for everyone.

📖 SVG Master 사용 설명서

웹에서 바로 벡터 그래픽을 디자인하고 코드를 생성하세요.

⌨️ 단축키

기능 단축키 설명
선택 V 객체 선택 및 이동
그리기 D 자유 곡선 그리기 (Pen)
지우개 E 클릭: 삭제 / 드래그: 문질러 삭제
페인트 B 클릭하여 색 채우기
사각형 R 드래그하여 사각형 생성
원 C 드래그하여 원 생성
삼각형 T 클릭하여 삼각형 생성
그룹화 / 해제 Ctrl+G 묶기/풀기
프로젝트 저장 Ctrl+S 작업 원본 저장 (.json)
프로젝트 열기 Ctrl+O 작업 불러오기
이미지 내보내기 Ctrl+Shift+S PNG 이미지 저장
복사 / 붙여넣기 Ctrl+C / V 복제
실행 취소 Ctrl+Z / Y 되돌리기 / 재실행
삭제 Del 선택 삭제
전체 선택 Ctrl+A 모두 선택
순서 변경 [ / ] 맨 뒤로 / 맨 앞으로
확대/축소 Wheel 마우스 휠 스크롤
화면 이동 Space + Drag Space (누른채) + 드래그

💡 유용한 팁

  • 미세 조정: 방향키로 1px씩 이동 (Shift: 10px)
  • 역방향 편집: 코드창을 수정하면 캔버스에 즉시 반영됩니다.
